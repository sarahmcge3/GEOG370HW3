In this project I learned how to display images in different projections

Describe in your own words how you displayed the map in different projections using QGIS

After importing the map as a vector layer in QGIS, I was then able to display the map with different projections by navigating to the project properties. In that tab I then chose what coordinate reference system I wanted to use to display the map and then applied it to the vector. This then manipulated the map in a multitude of ways as the different projections appear differenly depending on what aspect of the globe they compromise to make the map into a 2-dimensional object.
